# AI Student Academic Performance Predictor

This project is a PHP + MySQL web application for an academic performance predictor system. It includes:

- Student profile management
- Teacher and admin grade encoding
- Attendance monitoring
- Performance prediction
- Dashboard views for admin, teacher, and student users
- Report generation and notifications

## Dataset source
The sample dataset is adapted from the public UCI Student Performance dataset by Paulo Cortez and A. Silva (CC BY 4.0).

## How to run with XAMPP
1. Start Apache and MySQL in XAMPP.
2. Place this folder inside the XAMPP htdocs directory, for example:
   - C:\xampp\htdocs\final-project-group-1
3. Open the browser and go to:
   - http://localhost/final-project-group-1/
4. Use the default credentials:
   - Admin: admin / password123
   - Teacher: teacher / password123
   - Student: student / password123
